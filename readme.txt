FEATURES INCLUDED:
*   Instance Variable Method Generator
*   Recursive Partname finder & method generator
*   Enhanced Class Hierarchy Browser with:
    *   Additions to rename classes
    *   Additions to move classes
    *   Search the entire system for user specified
        implementors & senders
